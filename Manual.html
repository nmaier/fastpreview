FastPreview Manual

A PicaView replacement.

About

FastPreview is build to mimic old PicaView by ACDSystems. (mTools sucks :p). It has two components:

  • The Viewer - built do easily display images (at different zoom states)
  • The Shell Extension - it integrates with the regular Windows Contextual File Menu and displays wherever possible.

Features

The features are pretty limited. As intended!

Viewer

  • Best Fit-Zoom
  • "Free" zooming (10%-wise)
  • Auto-Refreshing files upon changes
  • Re-sampling display filters
  • Lossless JPEG transformation
    • Rotations
    • Flipping (Mirror-effects)
  • Adds the Viewer to the Open with list for supported types, so you can easily open the viewer from there, or even make it the default application.

Shell Extension

  • Configurable Thumbnail dimensions
  • Disable the in-menu Thumbnails altogether if you just want the other stuff.
  • Adds a page to the File Properties dialog detailing the image meta-data.
  • Adds a thumbnail maker component to the system, so that Explorer, etc. will be able to show thumbnails for supported file files, e.g. WebP.

Requirements

Minimal configuration

  • Windows Vista or later - and I indeed mean this! XP is dead — Get over it ;)
  • CPU that supports SSE3 instructions (64-bit version only)

Misc. stuff

  • You might have noticed by now that I hate writing manuals...
  • Pretty basic application in C/C++ without any frameworks (besides FreeImages and the XMP stuff).
  • Compiled using the Intel Compiler 14. If you're on AMD and the performance sucks, complain to Intel, not me ;(
  • There was a Version 1 developed in Borland C++Builder 6. Good times :p
  • Made in Germany — no, really!
  • Free in use, i.e. actually free; I don't monetize this application at all.
  • No support-ware!
